# スムレコ 配達効率計算 (SMOOTH RECORDER)

フードデリバリー配達員向けのシンプルで使いやすい売上・効率計算ツールです。
ブラウザ上で動作し、各プラットフォームでの稼働時間、配達件数、配達報酬などを入力することで、時給や配達単価などの効率データを自動計算します。

## 機能
- **複数プラットフォーム対応**: Uber Eats, 出前館, ロケットナウ, menu, その他（旧Wolt）の同時計算に対応。
- **時間の入力方法**:
  - 時刻での入力（開始時刻と終了時刻）
  - 時間での直接入力（日をまたいだり長時間の稼働を集計する際に便利）
- **詳細な内訳表示**:
  - 総売上における時給・単価だけでなく、「配達報酬のみ」をベースにした効率も同時計算してカッコ内に表記。
  - プロモーション（ボーナス）やチップによる変動を除いた純粋な配達ペースを把握できます。

## 使い方
1. ダウンロードまたはクローンして、`index.html`をブラウザで開きます。
2. 上部のトグルで「稼働時間」の入力方法（時刻または直接）を選択し、数値を入力します。
3. 稼働したプラットフォームのボックスに、件数・報酬・ボーナス・チップを入力します。
4. 一番下の「計算」ボタンを押すと、全体およびプラットフォーム別の時給や単価が下部に表示されます。
5. 上部に追従するナビゲーションバーをタップすることで、入力したいプラットフォームの欄に素早くスクロール可能です。

## ファイル構成
- `index.html`: アプリ本体（HTML/CSS/JS がすべて含まれています）
- `title-image.png`: ヘッダー用ロゴ画像

*(※`.bak` 拡張子がついているファイルは、過去の不要なバックアップデータです。)*

## 技術構成
- HTML/Vanilla CSS/Vanilla JavaScriptのみで作成され、通信を一切行わないスタンドアローンなSPA（シングルページアプリケーション）です。サーバーサイドでのデータ保存は行いせん。
